# Executive Superstore Sales Performance Dashboard

An interactive Power BI Executive Dashboard designed to analyze and extract actionable business insights from global superstore sales data. This project focuses on tracking enterprise revenue trends, evaluating regional performance, and understanding product category dynamics for data-driven executive decision-making.

### Dashboard Previe

###  Project Overview
The core objective of this executive-level dashboard is to transform raw sales data (`train.csv`) into a high-level visual story that helps stakeholders and management monitor business health. The report provides a comprehensive, drill-down view of how different regions, product segments, and shipping methods impact overall revenue.

### Executive Insights & Features
* **Sales Analytics & High-Level KPIs:** Visualizes revenue growth, total sales, and historical trends over the years to identify peak financial quarters.
* **Product Performance Dynamics:** Breaks down performance across major categories (Furniture, Office Supplies, Technology) and sub-categories to isolate top revenue drivers.
* **Geographical Mapping for Leadership:** Highlights high-performing US States and Regions (West, East, Central, South) to uncover regional market expansion opportunities.
* **Logistics Optimization:** Analyzes how different shipping modes (Standard Class, First Class, Same Day, etc.) impact sales volume and delivery trends.

---

###  Tools Used
* **Power BI Desktop:** For data modeling, DAX calculations, and executive-level interactive report visualization.
* **Dataset:** Superstore Sales Dataset (sourced from Kaggle).

###  How to View the Project
1. Clone or download this repository to your local machine.
2. Download the `.pbix` file.
3. Open the file using **Power BI Desktop** to interact with the executive filters, slicers, and charts.
